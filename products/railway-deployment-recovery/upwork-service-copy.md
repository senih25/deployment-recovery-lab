## Title options

## Short overview

I help founders and small teams recover broken Railway deployments without guesswork.

Typical issues include:

- `npm ci` / lockfile failures
- missing `package.json` or wrong root path
- invalid custom Railpack or build overrides
- app not responding / `502 Bad Gateway`
- build succeeded but runtime still broken

## What I deliver

- root cause summary
- minimum safe fix plan
- verification checklist
- clear next steps

## What I need from you

- repo link
- deploy/build logs
- screenshots if available
- current error summary

## What this is not

- full DevOps rebuild
- large refactor
- ongoing 24/7 maintenance
